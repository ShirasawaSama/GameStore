# Game Store

A simple homework for my Full Stack Developer course.

## Tech Stack

- Flask
- MongoDB
- Docker
- Docker Compose
- Angular
- Angular Material

## Installation

### Pre-requisites

- Docker
- Docker Compose
- Nodejs

### Steps


```bash
git clone https://github.com/ShirasawaSama/GameStore.git

cd GameStore
```

#### Frontend

```bash
cd frontend

npm install

npm run build
```

#### Backend

```bash
docker-compose up -d
```
